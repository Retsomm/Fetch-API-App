# Pesticide Public Opinion Mailbox FAQ App

This is a React Native application designed to display frequently asked questions from the Pesticide Public Opinion Mailbox. Users can search for questions and view detailed information using the application.

## Installation

Before you get started, make sure you have Node.js and npm installed. Follow these steps to install and run the application:

1. Download or clone this project to your local machine.

2. Open your terminal and navigate to the project directory.

3. Install the dependencies by running the following command:

   ```bash
   npm install

````

1. Next, start the application:

   ```
   npm start
   ```

   This will launch the development server for your React Native app.

2. You can then use the [Expo](https://expo.io/) client app to scan the QR code and run the application on your mobile device.

## **Features**

- The main screen displays a list of questions that can be searched by entering keywords.

- Clicking on an item in the question list will display detailed information, including the question category, question content, and answer.

## **Tech Stack**

- React Native: JavaScript framework for building cross-platform mobile apps.

- React Navigation: Handles app navigation and routing.

- Redux: State management.

- Expo: Accelerates React Native development.

- Data Source: Data is sourced from the [Pesticide Public Opinion Mailbox](https://data.moa.gov.tw/Service/OpenData/TactriMbox.aspx) API.
